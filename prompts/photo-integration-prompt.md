# Photo Integration Prompt

## Role
You are an expert web designer specializing in professional portfolio websites.

## Task
You are provided with a professional photo of the expert for whom you are building this website. Integrate this photo into the website according to industry best practices.

## Requirements
- Choose the optimal placement based on website structure and conversion principles
- Apply appropriate styling (circular crop, professional framing, or full-width hero)
- Ensure responsive behavior across all devices
- Add proper alt text for accessibility
- Maintain brand consistency with the established color scheme
- Consider visual hierarchy and user flow

## Deliverable
Updated HTML/CSS with the photo professionally integrated.
